## Projeto Final - Crimes em SP

Análise e manipulação de dados referentes a crimes no estado de São Paulo a partir dos conhecimentos adquiridos durante o Curso de Python  FEA.dev a fim de testar hipóteses e inferir fatos que serão apresentados publicamente na conclusão do curso.

## Listas:

* Atribuição e estruturas de variáveis
* Condicionais
* Loops
* Funções
* Numpy
* Pandas I
* Pandas II
* Visualização de dados
